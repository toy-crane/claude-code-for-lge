# 세션 정리 커스텀 커맨드 생성 계획

## Context
현재 세션에서의 중요한 의사결정 과정과 앞으로 해야할 작업들을 `.claude/sessions/`에 자동 정리하는 커스텀 커맨드를 만든다.

## 생성할 파일

### `.claude/commands/done.md`

커맨드 내용:
- 현재 세션의 대화 내용을 분석하여 다음을 정리
  - **의사결정 기록**: 세션에서 내린 주요 결정들과 그 이유
  - **완료한 작업**: 세션에서 실제로 수행한 작업 목록
  - **다음 할 일**: 앞으로 진행해야 할 작업들
- 결과를 `.claude/sessions/YYYY-MM-DD-주요내용요약.md` 파일로 저장 (예: `2026-02-25-todo-앱-카테고리-기능-추가.md`)
- 한글로 작성

### 출력 파일 형식 (`.claude/sessions/YYYY-MM-DD-주요내용요약.md`)

```markdown
# 세션 요약 - 주요 내용 요약 제목

## 주요 의사결정
- 결정 내용과 이유

## 완료한 작업
- 수행한 작업 목록

## 다음 할 일
- [ ] 앞으로 해야 할 작업들
```

## 수정할 파일
- 없음 (새 파일만 생성)

## 검증 방법
- `/done` 명령어 실행하여 `.claude/sessions/` 디렉토리에 마크다운 파일이 정상 생성되는지 확인
