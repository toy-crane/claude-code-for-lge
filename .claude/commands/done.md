현재 세션의 대화 내용을 분석하여 세션 요약을 작성해줘.

다음 항목을 정리:
- **주요 의사결정**: 세션에서 내린 주요 결정들과 그 이유
- **완료한 작업**: 세션에서 실제로 수행한 작업 목록
- **다음 할 일**: 앞으로 진행해야 할 작업들

결과를 `.claude/sessions/YYYY-MM-DD-주요내용요약.md` 파일로 저장해줘.
- 파일명의 날짜는 오늘 날짜, 요약은 핵심 키워드를 하이픈으로 연결 (예: `2026-02-25-todo-앱-카테고리-기능-추가.md`)
- 한글로 작성

출력 파일 형식:

```markdown
# 세션 요약 - {주요 내용 요약 제목}

## 주요 의사결정
- 결정 내용과 이유

## 완료한 작업
- 수행한 작업 목록

## 다음 할 일
- [ ] 앞으로 해야 할 작업들
```
