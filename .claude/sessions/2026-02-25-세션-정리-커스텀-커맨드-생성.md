# 세션 요약 - 세션 정리 커스텀 커맨드 생성

## 주요 의사결정
- `/done` 커스텀 커맨드를 `.claude/commands/done.md`에 생성하기로 결정 — 세션 종료 시 대화 내용을 자동으로 정리하여 지식을 보존하기 위함
- 세션 요약 파일을 `.claude/sessions/YYYY-MM-DD-주요내용요약.md` 형식으로 저장하기로 결정 — 날짜 기반 정렬과 내용 파악이 동시에 가능하도록
- 기존 `review.md` 커맨드의 간결한 스타일을 참고하여 `done.md` 작성

## 완료한 작업
- `.claude/sessions/` 디렉토리 생성
- `.claude/commands/done.md` 커스텀 커맨드 파일 생성
- `/done` 명령어 실행 테스트 (현재 세션)
